# ListModifyTransition 列表CURD过渡

### 介绍

基于`transition-group`封装的列表动画组件，当内部列表发生CURD时，会触发对应的移动动画

### 引入

此组件引入即可使用

```typescript
import { AwListModifyTransition } from 'sorarain';
```
