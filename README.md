<h1 align="center">组件库脚手架</h1>

<p align="center">基于vue3与typescript的Monorepo结构组件库框架</p>

## 灵感来源

[技术来源-掘金文章](https://juejin.cn/post/7117886038126624805)



## 功能

- 👆 Vue3 
- 💪 TypeScript
- 🌍 Vite
- 🍭 Less
- 📖 Eslint + Prettier



### 依赖安装

- npm i pnpm esno  -g
- pnpm i



### 打包

进入`packages/components`文件夹，执行

> pnpm run build
