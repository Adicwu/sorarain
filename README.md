### 介绍

Sorarain 是一个**功能性 Vue3 组件库**，于 2022 年开源。

目前只是一个面向web端的不成熟框架

## 特性

- 🚀 组件均为功能性组件，不涉及过多样式
- 🚀 零外部依赖，不依赖三方 npm 包
- 💪 使用 TypeScript 编写，提供完整的类型定义
- 🍭 支持按需引入和 Tree Shaking
- 🍭 支持服务器端渲染
<!-- - 🚀 性能极佳，组件平均体积小于 1KB（min+gzip） -->
<!-- - 💪 单元测试覆盖率超过 90%，提供稳定性保障 -->
<!-- - 📖 提供丰富的中英文文档和组件示例 -->

## 安装

在现有项目中使用 Sorarain 时，目前仅可通过 `yarn` 进行安装：

```bash
# 通过 yarn 安装
yarn add sorarain
```

## 快速上手
Sorarain开箱即用，请按照文档导入即可使用


## 浏览器支持
